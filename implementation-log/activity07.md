# Activity 7: Service Request Management

## Objective

Develop the Service Request module to allow employees to request IT services, hardware, software, and access permissions.

---

## Business Requirement

Employees frequently request enterprise services such as laptops, software installations, VPN access, email accounts, and hardware replacements. This module provides a centralized mechanism for submitting and tracking such requests.
