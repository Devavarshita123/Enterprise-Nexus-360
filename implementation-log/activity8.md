# Activity 10: Employee Onboarding Module

## Objective

Develop the Employee Onboarding module to automate the provisioning process for newly hired employees.

## Business Requirement

Every new employee requires IT resources before joining. This module centralizes onboarding requests and will later trigger automated provisioning workflows.
