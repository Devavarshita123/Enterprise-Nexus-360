# Activity 8: Procurement Management Module

## Objective

Develop the Procurement Management module to manage enterprise purchase requests from initiation to approval.

## Business Requirement

Organizations frequently procure IT assets such as laptops, monitors, software licenses, and networking equipment. A structured procurement process ensures transparency, accountability, and approval compliance.

## Table Created

**ENX Procurement Request**

## Fields

- Request Number
- Requested By
- Department
- Vendor
- Asset Type
- Quantity
- Estimated Cost
- Priority
- Requested Date
- Required By
- Status
- Manager Approval
- Finance Approval
- Procurement Approval
- Remarks

## Relationships

- Employee Profile
- Department
- Vendor

## Concepts Learned

- Enterprise Procurement Workflow
- Reference Fields
- Choice Fields
- Data Relationships

## Outcome

Successfully designed the Procurement Request module that will later integrate with approval workflows, notifications, asset creation, and vendor management.
