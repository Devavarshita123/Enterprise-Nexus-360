Dynamic Employee Widget

## Objective
Developed a dynamic Service Portal widget to display employee information.

## Components Implemented
- HTML Template
- Server Script
- Client Controller

## Functionality
- Retrieved employee records from the ENX Employee table.
- Calculated the total employee count.
- Displayed the count dynamically on the portal.

## Concepts Learned
- Service Portal Widget Architecture
- Server Script
- Client Controller
- Data Binding
- AngularJS Expressions

## Result
Successfully created a dynamic widget that displays real-time employee statistics.
